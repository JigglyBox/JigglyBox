Welcome to my GitHub profile! I'm a hands-on developer with a strong foundation in **C++**, **game development**, and **engine tooling**. ## 🧩 Highlight Projects

### 🧿 Spirit Ward

> A 3D third-person shooter set in olden-day Singapore, where you play as an exorcist defending kampungs from malevolent spirits using a Shakujo, soul and energy casting, and powerful totems.
>
> My work focused on engine tools and team integration: console functionality, logging systems, undo/redo using the command pattern, animation graph implementation, UI/UX design, gameplay loop planning, and design documentation.

### 🔥 BoneFire Mayhem

> A 2D game built on a custom engine, with C++ to C# scripting integration, gameplay UI, win/lose screens, timer logic, and layered asset systems.
>
> I helped marshal C++ engine code into a C# scripting layer through a Mono-style setup, making lower-level engine systems usable from gameplay scripts.
>
> [Play on itch.io](https://buburito.itch.io/bonefire-mayhem)

### 🌙 Goodnight, Homepage

> A tiny interactive webpage that gets progressively sleepier over time, built with HTML, CSS variables, and JavaScript timers.
>
> JavaScript tracks the page's sleepiness, updates the message and title, and changes CSS properties like brightness, blur, and rotation. A coffee button wakes the page back up a little, because even a webpage deserves emotional support and caffeine.
